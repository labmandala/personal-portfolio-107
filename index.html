Holstee Manifesto


HTML and CSS Creative Exercise

Individual rendition of selections from the famous Holstee Manifesto. This is a version of one of my first hands-on coding exercises where I had to figure out positioning of the blocks, allowing me to put newly learned code into practice. I was also introduced to color theory, experimenting with different color schemes. First I recreated the specific layout into code, then later added more advanced CSS techniques, using grid properties to create a fully responsive version.
